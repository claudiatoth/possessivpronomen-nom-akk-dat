# Audio — Possessivpronomen Nom + Akk + Dat

50 WAV generate automat cu Microsoft Hedda Desktop (System.Speech) — voce germană nativă, rate -2.

## Structură

- **18 WAV teorie** (`p-*`, `cap-*`) — forme brute, paradigmă cheie, capcane sein/ihr, capcane euer
- **32 WAV flashcards** (`fc-*`) — câte unul pentru fiecare card

## Categorii

- `p-mein/dein/sein/ihr/unser/euer.wav` — cele 6 forme brute Nominativ m/n
- `p-meinen/meinem/meiner/meine.wav` — paradigmă cheie pentru tabel
- `cap-sein-florian / cap-ihr-carolina / cap-es-kind / cap-sie-pl.wav` — capcana sein vs ihr
- `cap-euer-vater / cap-eure-mutter / cap-euren-bruder / cap-ihr-auto.wav` — capcana euer + Sie politicoasă
- `fc-*.wav` — cele 32 flashcards (forme brute, Akk masculin, Dat m/n/f/pl, capcane)

## Regenerare

Script: `C:\Users\user\AppData\Local\Temp\generate-audio-possessiv.ps1`
